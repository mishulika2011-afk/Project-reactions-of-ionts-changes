#  Реакції йонного обміну між електролітами у водних розчинах

## Дослід 1. Взаємодія хлоридної кислоти з натрій гідроксидом

**Молекулярне рівняння:**
```
HCl + NaOH → NaCl + H₂O
```

**Повне йонне рівняння:**
```
H⁺ + Cl⁻ + Na⁺ + OH⁻ → Na⁺ + Cl⁻ + H₂O
```

**Скорочене йонне рівняння:**
```
H⁺ + OH⁻ → H₂O
```

---

## Дослід 2. Взаємодія купрум(ІІ) сульфату з барій нітратом

**Молекулярне рівняння:**
```
CuSO₄ + Ba(NO₃)₂ → BaSO₄↓ + Cu(NO₃)₂
```

**Повне йонне рівняння:**
```
Cu²⁺ + SO₄²⁻ + Ba²⁺ + 2NO₃⁻ → BaSO₄↓ + Cu²⁺ + 2NO₃⁻
```

**Скорочене йонне рівняння:**
```
Ba²⁺ + SO₄²⁻ → BaSO₄↓
```

---

## Дослід 3. Взаємодія натрій карбонату з хлоридною кислотою

**Молекулярне рівняння:**
```
Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑
```

**Повне йонне рівняння:**
```
2Na⁺ + CO₃²⁻ + 2H⁺ + 2Cl⁻ → 2Na⁺ + 2Cl⁻ + H₂O + CO₂↑
```

**Скорочене йонне рівняння:**
```
CO₃²⁻ + 2H⁺ → H₂O + CO₂↑
```

---

## Дослід 4. Взаємодія купрум(ІІ) сульфату з натрій гідроксидом

**Молекулярне рівняння:**
```
CuSO₄ + 2NaOH → Cu(OH)₂↓ + Na₂SO₄
```

**Повне йонне рівняння:**
```
Cu²⁺ + SO₄²⁻ + 2Na⁺ + 2OH⁻ → Cu(OH)₂↓ + 2Na⁺ + SO₄²⁻
```

**Скорочене йонне рівняння:**
```
Cu²⁺ + 2OH⁻ → Cu(OH)₂↓
```

---

## Дослід 5. Якісні реакції на йони

### а) Якісна реакція на сульфат-йон

**Молекулярне рівняння:**
```
Ba(NO₃)₂ + Na₂SO₄ → BaSO₄↓ + 2NaNO₃
```

**Повне йонне рівняння:**
```
Ba²⁺ + 2NO₃⁻ + 2Na⁺ + SO₄²⁻ → BaSO₄↓ + 2Na⁺ + 2NO₃⁻
```

**Скорочене йонне рівняння:**
```
Ba²⁺ + SO₄²⁻ → BaSO₄↓
```

### б) Якісна реакція на хлорид-йон

**Молекулярне рівняння:**
```
AgNO₃ + NaCl → AgCl↓ + NaNO₃
```

**Повне йонне рівняння:**
```
Ag⁺ + NO₃⁻ + Na⁺ + Cl⁻ → AgCl↓ + Na⁺ + NO₃⁻
```

**Скорочене йонне рівняння:**
```
Ag⁺ + Cl⁻ → AgCl↓
```
